# Audio Assets

## gpu.mp3
Place your `gpu.mp3` file in this directory. The audio file should be:
- Ambient/atmospheric music that complements the AI visualization
- Suitable for continuous looping
- Optimized for web (reasonable file size)

The audio will:
- Start playing when user clicks "Enter Visualization" 
- Loop continuously while the data sculpture is active
- Be controllable via mute/unmute button
